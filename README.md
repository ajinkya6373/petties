# Petties

[Petties](https://petties.vercel.app/) - An E-Commerce website for pets.

https://github.com/ajinkya6373/petties/blob/master/public/assets/petties.mp4


## How to install and run locally?

```
$ git clone https://github.com/ajinkya6373/petties
$ cd petties
$ npm install
$ npm start
```

## Features -

- Landing page with a list of featured categories
- Product listing page
- Filter and sort products functionalities
- Search product functionality
- Individual product page
- Cart Managment
- Wishlist managment
- Address management
- Order management
- Order Detail page
- Authentication:
  - User Signup
  - User Login
  - logout page
- Error Page
- Customized Toast
-  Loaders
- Responsive

## Built with -

- React JS
- React Context API + useReducer
- React Router v6
- Styled Component
- Exprees,Node,Mongodb for backend
